# QuakeSafe Native (Flutter)

This is the native version of QuakeSafe built with Flutter.

## How to Run

Since the project structure was generated manually, you must initialize the platform folders (Android/iOS) first.

1.  **Install Flutter**: [Download & Install](https://flutter.dev/docs/get-started/install)
2.  **Initialize Project**:
    ```bash
    # Open terminal in this folder
    flutter pub get
    flutter create . --platforms=android
    ```
3.  **Run**:
    ```bash
    flutter run
    ```

## Features
- Native Performance
- Tactical "Dark Mode" UI
- SOS Strobe & Haptic Feedback
- Offline-first architecture
